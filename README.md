## 快速入门

- [如何开始使用？](https://ng-alain.com/version/9.x/docs/getting-started/zh)。

### 启动项目

1. `npm install` 安装项目依赖

2. `npm run dev` 已开发模式启动项目

### 打包部署上线

1. `npm run build:prod` 生成 dist 包，拷贝 dist 到服务器
